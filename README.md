# Seb-Discord-Bot
Seb is a custom-built automation bot designed for Discord communities. It integrates with the Discord API and Twitch API to provide automated announcements, command handling, and live stream notifications.<br/>

This repo contains a public release version of the bot. Some internal or experimental features are not included in this version.

To run this bot, you must:
Create a Discord Bot Application via the Discord Developer Portal <br/>
Generate a bot token<br/>
Register a Twitch Developer Application<br/>
Add your API credentials to the config/env files<br/>
